## What is ocrbase?

ocrbase is an open-source platform for converting documents into structured data. It combines state-of-the-art OCR (PaddleOCR-VL-1.5) with LLM-powered extraction to turn PDFs and images into typed JSON.

### How it works

<Steps>
  <Step>
    ### Parse

    Upload a PDF or image. ocrbase converts it to clean markdown using vision-language OCR.

  </Step>
  <Step>
    ### Extract

    Provide a schema describing your desired output. An LLM extracts structured JSON from the parsed markdown.

  </Step>
  <Step>
    ### Use

    Get typed, validated data ready for your application.

  </Step>
</Steps>
