## Setup

The SDK reads your API key from `OCRBASE_API_KEY` automatically:

```ts
import { parse } from "ocrbase";

const { text } = await parse("./invoice.pdf");
console.log(text);
```

<Callout type="info" title="Environment variable">
  The `apiKey` defaults to the `OCRBASE_API_KEY` environment variable if not
  provided.
</Callout>

## Parse a document

Convert a PDF to markdown:

```ts
import { parse } from "ocrbase";

const { text } = await parse("./invoice.pdf");

console.log(text);
// # Invoice #1234
// Date: 2024-01-15
// ...
```

## Extract structured data

Define a schema and extract typed JSON:

```ts
import { extract } from "ocrbase";

const { object } = await extract("./invoice.pdf", {
  vendor: "string",
  invoiceNumber: "string",
  total: "number",
  lineItems: [
    {
      description: "string",
      quantity: "number",
      price: "number",
    },
  ],
});

console.log(object);
// {
//   vendor: "Acme Corp",
//   invoiceNumber: "INV-1234",
//   total: 1420.00,
//   lineItems: [
//     { description: "Widget A", quantity: 10, price: 42.00 },
//     { description: "Widget B", quantity: 5, price: 200.00 }
//   ]
// }
```

## Supported file types

| Format | MIME Type         |
| ------ | ----------------- |
| PDF    | `application/pdf` |
| PNG    | `image/png`       |
| JPEG   | `image/jpeg`      |
| WebP   | `image/webp`      |
| TIFF   | `image/tiff`      |
| BMP    | `image/bmp`       |
