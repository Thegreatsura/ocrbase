## Quick start

```ts
import { extract, parse } from "ocrbase";

const { text } = await parse("./document.pdf");
const { object } = await extract("./invoice.pdf", {
  vendor: "string",
  total: "number",
});
```

<Callout type="info" title="Environment variable">
  `parse()` and `extract()` automatically use `OCRBASE_API_KEY` when `apiKey` is
  not passed.
</Callout>

## Methods

### `parse(file, options?)`

Convert documents to markdown via OCR. Returns the full text and per-page content.

```ts
import { parse } from "ocrbase";

const result = await parse("./document.pdf");

console.log(result.text); // Full markdown
console.log(result.pages); // Per-page content
console.log(result.metadata.pageCount); // Number of pages
```

[Full parse reference](/docs/sdk/parse)

### `extract(file, schema, options?)`

Extract structured data from documents using a schema.

```ts
import { extract } from "ocrbase";

const result = await extract("./invoice.pdf", {
  vendor: "string",
  total: "number",
});

console.log(result.object); // { vendor: "Acme Corp", total: 1420.00 }
```

[Full extract reference](/docs/sdk/extract)

### `createOcrBase(config)` (advanced)

Use this when you want a reusable client instance and explicit `{ data, error }`
results.

```ts
import { createOcrBase } from "ocrbase";

const ocr = createOcrBase({ apiKey: "sk_..." });
const result = await ocr.parse({ file: "./document.pdf" });

if (result.error) throw result.error;
console.log(result.data.text);
```

#### Configuration options

| Option                   | Type       | Default                   | Description              |
| ------------------------ | ---------- | ------------------------- | ------------------------ |
| `apiKey`                 | `string`   | `OCRBASE_API_KEY` env     | Your API key             |
| `baseUrl`                | `string`   | `https://api.ocrbase.dev` | Server URL               |
| `timeoutMs`              | `number`   | `300000` (5 min)          | Request timeout          |
| `createWebSocket`        | `function` | â€”                         | Custom WebSocket factory |
| `wsMaxReconnectAttempts` | `number`   | `5`                       | Max reconnect attempts   |

## Document input types

The SDK accepts multiple input types for the `file` argument:

```ts
// File path (Node.js / Bun)
await parse("./document.pdf");

// URL string
await parse("https://example.com/document.pdf");

// URL object
await parse(new URL("https://example.com/doc.pdf"));

// ArrayBuffer
await parse(arrayBuffer);

// Blob (browser)
await parse(blob);

// ReadableStream
await parse(readableStream, { fileName: "doc.pdf" });
```
